# Party Tank Prague — v5 Experience Prototype

Statický one-page prototyp pre nový brand Party Tank Prague.

## Štruktúra

```txt
index.html
style.css
main.js
.nojekyll
img/
  hero-night.svg
  interior-lounge.svg
  route-glow.svg
  og-preview.svg
```

CSS a JS sú v roote, nie v `assets/`.

## Čo je hotové

- moderný one-page vizuál
- vlastné SVG ilustrácie bez externých fotiek
- EN/CZ prepínač
- responzívny layout
- animácie, reveal efekty, marquee, interaktívne balíčky
- booking form vizuálne pripravený
- GitHub Pages ready

## Čo ešte nie je produkčné

- formulár zatiaľ reálne neodosiela
- telefón a e-mail sú dočasne podľa pôvodného Hummer webu
- finálne fotky / logo / doména sa majú doplniť až po schválení dizajnu

## GitHub Pages

Repo → Settings → Pages → Source: Deploy from branch → Branch: `main` → Folder: `/root`.
